# AutoSpecification

Python-утилита для автоматизации создания Excel-книг спецификаций и скрытия всех листов кроме "Specification"

**Что выполняет утилита:**
- Создаёт файлы спецификаций из исходных файлов инвойсов с названием `Invoice ... fsc`
- Работает по двум сценариям: создание спецификаций с типами `XLSX` или `XLS`
- Скрывает в Excel-книге все листы кроме `Specification`
- Позволяет работать с диапазонами номеров инвойсов (указывается в формате `1-15,16,17`)
- Сохранение пути к директории с инвойсами (если ранее уже был введен путь к директории, то в дальнейшем указывается только для изменения пути)
- Работа через консольное меню и проверка существования файлов

## Используемые библиотеки

- `openpyxl` — работа с XLSX
- `pywin32` — работа с Excel (экспорт в XLS через COM)
- `colorama` — цветной вывод в консоль

## Установка
- Скачайте все файлы из репозитория
- Для установки всех зависимостей используйте файл `Install requirements.bat` - он установит библиотеки из файла requirements.txt
